## Task Description
This project performs sentiment analysis on product review data using Natural Language Processing (NLP) techniques. 
The goal is to classify customer reviews into positive and negative sentiments and analyze overall customer feedback.

## Dataset
- Source: Kaggle (Cleaned Product Reviews Dataset)
- Columns used:
  - `cleaned_review_text`
  - `sentiment`

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression
- Matplotlib, Seaborn
- Jupyter Notebook

## Steps Performed
1. Loaded and explored the dataset
2. Selected required columns and handled missing values
3. Converted text data into numerical features using TF-IDF
4. Split data into training and testing sets
5. Trained Logistic Regression model
6. Evaluated model using accuracy and classification report
7. Tested model with custom user reviews
8. Visualized sentiment distribution

## Results & Insights
- The majority of reviews are positive
- Sentiment distribution graph shows customer opinion trends
- The model successfully predicts sentiment for new reviews
